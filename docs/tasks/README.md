# Tasks Documentation Index

This folder tracks everything related to the Tasks system: what exists today, what changed, known issues, automation details, and test coverage. Use it as the entry point when working on task features or debugging task automation.

## Contents
- `HISTORY.md` – chronological log of task-related changes and fixes.
- `ISSUES.md` – task-specific issues log (pulled from `docs/ISSUES_LOG.md` with extra context).
- `AUTOMATION.md` – current task API surface, automation behavior, telemetry, and tests.
- `TEST_PLAN_TASKS.md` – exhaustive test plan for tasks/automation (API, chat, UI, telemetry), including confidence chip UI check and priority/blocked/intent heuristics coverage.

- Global issues: `docs/ISSUES_LOG.md`.
