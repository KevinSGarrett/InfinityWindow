# 8) UI & Accessibility

**U-Proj-01..06** – Project flows (create/select/rename/delete/focus restore).  
**U-Conv-01..08** – Conversation flows (new chat, folder assign, drag/drop between folders).  
**U-Tasks-01..07** – Task panel CRUD + auto updates surface.  
**U-Docs-01..06** – Ingestion UI, progress cards, history table.  
**U-Files-01..07** – Files tree, diffs, apply/rollback.  
**U-Search-01..06** – Query input, result list, jump‑to‑doc.  
**U-Term-01..07** – Start/cancel, output streaming.  
**U-Usage-01..03** – Token/time counters.

**N-A11y-01** – Keyboard traversal across right column tabs.  
**N-A11y-02** – Visible focus ring for all actionable controls.  
**N-A11y-03** – ARIA roles/labels on tablist, tree, buttons.  
**N-A11y-04** – Contrast ratios (WCAG AA).  
**N-A11y-05** – Screen reader friendly labels for progress.  
**N-A11y-06** – Live region announcements for streaming.  
**N-A11y-07** – Avoid focus traps after dialogs.  
**N-A11y-08** – Resize to 1280×720 & high DPI.  
**N-A11y-09** – Localization safety (long strings).  
**N-A11y-10** – Error summary linkage.
