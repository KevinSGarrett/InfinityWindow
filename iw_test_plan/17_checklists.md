# 17) High‑Signal Checklists

## Daily Smoke (5–10 minutes)
- [ ] Backend `/health` OK; version matches docs
- [ ] Create a fresh project; select it
- [ ] Send a quick chat in `auto` mode; response streams
- [ ] Ingest SmallRepo; job completes; search returns the new docs
- [ ] Files panel opens and previews a text file
- [ ] Terminal runs `dir` and exits 0
- [ ] Telemetry shows correct increments

## Pre‑Release
- [ ] All S, A, U critical tests green
- [ ] Security K-SEC-10..22 passed in last 24h
- [ ] Perf regressions ≤ 20%
- [ ] Test report produced and archived
