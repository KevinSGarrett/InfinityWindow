# 18) Bug Bash Playbook

- Charter-based sessions (60–90m) by area: Chat, Ingestion/Search, Files/Terminal, UI Navigation.
- Use mnemonics: **CRUDS-FET** (Create‑Read‑Update‑Delete‑Search – Failure‑Edge‑Threats).
- Rotate personas: new user, power user, operator.
- Force errors: unplug network, fill disk, kill backend.
- Capture: video, console logs, `/debug/telemetry` snapshots at start/end.
